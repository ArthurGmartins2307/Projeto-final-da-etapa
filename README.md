📄 Documentação - SportLife Hub
1. Fluxo do Sistema 🔄

O fluxo do sistema no SportLife Hub é simples e intuitivo, com as seguintes etapas principais:

Página Inicial (Home) 🏠

O usuário entra no site e é apresentado a uma visão geral do SportLife Hub.

Exibição de informações como número de usuários ativos, número de treinos realizados e outros dados da plataforma.

O usuário pode navegar para as páginas principais: Quem Somos, Academias, Calcular Dieta e Lista de Treinos.

Quem Somos 👨‍👩‍👧‍👦

Exibição das informações sobre a equipe por trás da plataforma.

Esta página contém detalhes sobre cada membro da equipe (foto e descrição).

Busca de Academias 🏋️‍♂️

O usuário pode pesquisar academias filtrando por cidade e modalidade.

Uma lista de academias é exibida conforme os critérios de busca.

Calcular Dieta 🍏

O usuário preenche um formulário com informações como idade, peso, altura, nível de atividade física e objetivo (ex: emagrecer).

Após preencher os dados, o sistema calcula as calorias diárias necessárias, o IMC e os macronutrientes recomendados (proteínas, carboidratos e gorduras).

Lista de Treinos 💪

O usuário pode pesquisar e adicionar treinos à sua lista pessoal.

A lista de treinos mostra exercícios, suas descrições e permite ao usuário adicionar ou remover itens.

2. Funcionalidades ⚙️

As principais funcionalidades do sistema são:

Cálculo de Dieta 🍽️: O usuário insere dados como peso, altura e nível de atividade física para calcular a quantidade ideal de calorias e macronutrientes (proteínas, carboidratos e gorduras) necessários para atingir seu objetivo (como emagrecimento, ganho de massa, etc.).

Busca por Academias 🏋️: O usuário pode filtrar academias por cidade e modalidade (academia, pilates, yoga, etc.), visualizando os detalhes de cada uma.

Adição e Remoção de Treinos 📝: O usuário pode pesquisar treinos específicos (exercícios de musculação, etc.) e adicionar à sua lista de treinos pessoais.

Página de Equipe (Quem Somos) 👨‍👩‍👧‍👦: Detalhamento sobre os membros da equipe que desenvolveu a plataforma, incluindo fotos e descrições.

O site é totalmente interativo no cliente, com JavaScript manipulando as interações e exibindo os resultados no navegador.

3. Tecnologias Utilizadas 💻

Frontend:

HTML5 📄: Utilizado para a estrutura e marcação das páginas.

CSS3 🎨: Responsável pelo design e layout das páginas, incluindo a responsividade.

JavaScript (Vanilla JS) 💻: Usado para a lógica interativa do site, como o cálculo da dieta, a busca de academias e a manipulação da lista de treinos.

Outras Ferramentas:

GitHub 🔧: Utilizado para versionamento e hospedagem do código.

GitHub Pages 🌐: Usado para hospedar o site online de maneira estática.

Observação: O projeto não possui backend nem banco de dados, funcionando apenas no lado do cliente (navegador).

4. Como Executar o Projeto 🚀

Clonar o Repositório 🔄:

Abra o terminal (ctrl + j) e clone o repositório do GitHub:

git clone https://github.com/Daniel-Vara/Projetinho-FInal.git
cd Projetinho-FInal


Abrir o Projeto no Navegador 🌍:

Não há necessidade de instalar dependências ou rodar comandos de servidor. O projeto é totalmente estático.

Abra o arquivo index.html no seu navegador de preferência.

Testar as Funcionalidades 🧪:

Navegue pelas páginas: Home, Quem Somos, Busca de Academias, Calcular Dieta e Lista de Treinos.

Teste as interações (cálculo de dieta, adição de treinos à lista pessoal, etc.).

Hospedagem Online (opcional) 🌐:

Se desejar hospedar o projeto online, você pode usar o GitHub Pages (já configurado para este repositório).
